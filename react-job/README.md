### Intro
> 找工作 — 基于 React 16 的实时聊天单页应用

- React, Redux, React-router 4, redux-thunk, antd-mobile
- Socket.io, MongoDB, Express
- [Redux 实现](https://github.com/ifyour/my-redux), React 服务端渲染 SSR

### Usage
```bash
# 安装依赖
$ yarn

# 启动服务端
$ yarn run server

# 开发
$ yarn start

# 构建
$ yarn run build
```
